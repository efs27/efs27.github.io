Política de Privacidade

Última atualização: 19 de junho de 2025

Esta Política de Privacidade explica como os nossos aplicativos coletam, usam e compartilham informações quando você utiliza nossos aplicativos.

1. Coleta e Uso de Dados

Nosso aplicativo utiliza o Google AdMob para exibir anúncios. O AdMob pode coletar os seguintes dados:

  • Advertising ID (AAID): Identificador único para personalização de anúncios.
  • Dados de uso: Interações com anúncios (cliques, visualizações).
  • Dados técnicos: Tipo de dispositivo, sistema operacional e região aproximada.

Observação: Não coletamos dados pessoais como nome, e-mail ou informações de contato.

2. Compartilhamento de Dados

Os dados coletados pelo AdMob são compartilhados com:

  • Google: Para exibição e medição de anúncios (conforme a Política de Privacidade do Google).
  • Redes de publicidade terceirizadas: Parceiros do Google AdMob.

3. Como Desativar a Personalização de Anúncios

Você pode optar por desativar o uso do Advertising ID para anúncios personalizados:

  • No Android: Acesse Configurações > Google > Anúncios > Limpar ID de publicidade.

4. Alterações nesta Política

Podemos atualizar esta política periodicamente. Recomendamos que você revise esta página para se manter informado.
